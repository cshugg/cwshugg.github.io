auth_get_users

Authentication endpoint. Returns all online users. Returns 200 OK in any place.

auth_login

Login endpoint. Captures username+password and verifies against database.

auth_logout

Logout endpoint. Clears authentication token. Always returns 200 OK.

auth_user

Authentication endpoint. Recieves username and returns ip, port, and pub_key if it is a valid user. Returns 200 OK in any place.

debug_clear

Debugging endpoint. Clears all users and invalidates all JWT tokens. Requires "backdoor" god mode cookie. Returns 200 OK.

health_check

Basic healthcheck endpoint. Returns 200 OK.

hello
