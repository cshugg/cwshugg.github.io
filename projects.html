Personal Projects

I like coding so much that I do it in my free time! Here are a few projects I've worked on in the past:

WRIVCam

Dash cams are expensive, so I made my own. Struggling to come up with a cool name for it, I ended up with WRIVCam: "Wooden Ragtag In-Vehicle Camera." It's comprised of a Raspberry Pi 3B+, some jumper cables, a few LEDs, two buttons, and a Raspberry Pi camera module, all living in a small wooden box I bought from A.C Moore.

It's driven by some Python code I wrote for it. You can find the code, and more details, on my GitHub Page.

ttydo

I do quite a lot of work on the Linux command-line, and I like keeping track of my "to-dos." So I created ttydo to keep track of them straight from the command-line. It supports the creation of multiple "task lists," each containing their own "tasks" that describe something that needs doing. Tasks can be added, deleted, edited, reordered, etc. I've been using it myself and have found it pretty useful.

I wrote it entirely in vanilla C code. Check out its GitHub Page.
